# 🗑️ Bloatware Removers
> Utilities focused on uninstalling preinstalled apps, removing Edge/OneDrive/Cortana, and cleaning UI clutter.

### [Blackbird](https://www.getblackbird.net/)
- **Type:** `Portable EXE`
- **OS:** `Both`
- **Last Verified:** `2026-04-23`
- **Description:** One-click optimizer for removing bloatware and tuning performance
- **Launch/Install:** Download ZIP, run executable
- **Notes:** Auto-creates restore point, aggressive defaults

### [ChrisTitusTech/winutil](https://github.com/ChrisTitusTech/winutil)
- **Type:** `PowerShell GUI`
- **OS:** `Both`
- **Last Verified:** `2026-04-23`
- **Description:** All-in-one toolkit with interactive bloatware removal tab
- **Launch/Install:** `irm https://christitus.com/win | iex`
- **Notes:** Most popular, allows selective app removal

### [Fs00/Win10BloatRemover](https://github.com/Fs00/Win10BloatRemover)
- **Type:** `PowerShell`
- **OS:** `Both`
- **Last Verified:** `2026-04-23`
- **Description:** CLI-focused script for bulk app and service removal
- **Launch/Install:** Execute `.ps1` elevated
- **Notes:** Fast, includes rollback commands

### [meetrevision/playbook](https://github.com/meetrevision/playbook)
- **Type:** `Ansible/PowerShell`
- **OS:** `Both`
- **Last Verified:** `2026-04-23`
- **Description:** Automated deployment scripts for clean Windows images
- **Launch/Install:** Run playbook via Ansible or manual script
- **Notes:** Enterprise-focused, removes OEM bloat pre-deploy

### [memstechtips/UnattendedWinstall](https://github.com/memstechtips/UnattendedWinstall)
- **Type:** `Configuration Scripts`
- **OS:** `Win11`
- **Last Verified:** `2026-04-23`

- **Launch/Install:** Integrate into ISO media
- **Notes:** Requires ISO rebuilding; prevents bloat from ever installing
