# 🔒 Telemetry Blockers
> Tools specifically designed to disable diagnostic data collection, tracking services, and Microsoft analytics pipelines.

### [DoNotSpy10](https://pxc-coding.com/donotspy10/)
- **Type:** `Portable EXE`
- **OS:** `Win10`
- **Last Verified:** `2026-04-23`
- **Description:** Toggle-based manager for disabling Windows telemetry endpoints
- **Launch/Install:** Download, run as admin
- **Notes:** Basic UI, focuses on registry/service toggles

### [O&O ShutUp10++](https://www.oo-software.com/en/shutup10)
- **Type:** `Portable EXE`
- **OS:** `Both`
- **Last Verified:** `2026-04-23`
- **Description:** Comprehensive antispy utility with 200+ telemetry controls
- **Launch/Install:** Download EXE, run
- **Notes:** Safe presets available, highly recommended for beginners

### [privacy.sexy](https://privacy.sexy/)
- **Type:** `Web Generator`
- **OS:** `Both`
- **Last Verified:** `2026-04-23`
- **Description:** Generates transparent PowerShell scripts to disable telemetry services
- **Launch/Install:** Configure online, export `.ps1`, run as admin
- **Notes:** Open-source, audit-friendly, no cloud execution
